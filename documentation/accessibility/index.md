# Accessibility Guidelines

Accessibility standards and compliance requirements.

## Standards

- **WCAG 2.1 Level AA** — Target compliance level for all koad:io products
- **Color Contrast:** Minimum 4.5:1 for body text, 3:1 for large text
- **Keyboard Navigation:** All interactive elements keyboard accessible
- **Semantic HTML:** Proper heading hierarchy, form labels, landmark regions
- **ARIA Attributes:** Used when semantic HTML is insufficient

## Key Areas

- **Color:** Contrast ratios, colorblind-safe palettes
- **Typography:** Text size, line height, line length for readability
- **Interaction:** Focus indicators, keyboard shortcuts, touch targets (minimum 44x44px)
- **Motion:** Respecting prefers-reduced-motion, avoiding autoplaying animations
- **Forms:** Clear labels, error messages, validation feedback
- **Images:** Alt text, SVG accessibility, decorative image handling

## Tools

- Lighthouse (Chrome DevTools) — automated accessibility scoring
- axe DevTools — accessibility testing
- WAVE browser extension — visual accessibility feedback
- WebAIM contrast checker — color contrast verification
